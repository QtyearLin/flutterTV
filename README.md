# flutterTV

由于Google官方并没有退出TV版Flutter SDK，所以用Flutter尝试编写TV应用，主要是焦点框和焦点的处理，其他的和手机应用差别不大。焦点框和焦点处理比较麻烦，所以Flutter的TV应用开发还不成熟，体验到时很流畅，开发成本比较高，还是用原生leanback开发可能要快一些。
